# 7. number_puzzle

## Goal

1. Put the numbers 1 through 9 in a 3x3 grid.
2. 9 is marked with *.
3. Randomly shuffle the numbers when the game starts.
4. The player arranges the numbers in order using the arrow keys on the keyboard.
